# Campsite

An AI-powered analysis assistant for Jupyter notebooks. Campsite provides an interactive chat interface for hypothesis generation and analysis code generation based on loaded data.

## Installation

```bash
pip install -e .
npm install
node esbuild.config.js
```

## Development

```bash
# Install in development mode
pip install -e .

# Build frontend
npm install
node esbuild.config.js

# Run tests
pytest tests/test_campsite_lib.py -v
```
